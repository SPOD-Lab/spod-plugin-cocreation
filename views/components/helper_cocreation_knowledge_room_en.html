Cocreation Knowledge room helper
The Cocreation knowledge room is meant to stimulate reflection on Open Data through three steps process of co-writing document (with a template) for Explore, Ideas, and Outcome>. Datalets can be created and annotated via post-it like comments (notified in real time to participants).
1
Mode

This button, on the upper left corner, allow to select:

  • Data
  • Co-Creation
2
Data

All members can add datasets from open data portal (e.g., @CKAN), these are visible as a list in the Bookshelf list:

  • Bookshelf shows the datalets inserted in the list.
  • Add dataset allows to insert any dataset into the Bookshelf. A wizard will help you to add a the dataset.

3
Co-creation

In this room , on the left side a collaborative space allows to edit documents collaboratively in real-time and provides a list of features like:

  • Clear collaboration: it is possible to see changes being made by other users in real time and each participant is identified by a color and name.
  • Chat: a synchronous communication is available with other people who are using the document.
  • Recording Revisions; it is possible to save specific versions at any time. A time slider shows anyone the history of the shared document.
  • Exporting: a document can be downloaded in plain text, HTML, Open Document, Microsoft Word or PDF format.

On the right side, it is possible to create (a list of) datalets from scratch () or from MySpace (). It is also possible to add notes () on top of each datalet (Post-it like).

4
Co-creation: features

On the right side, a list of features are available:

  • User Interface allows to enlarge, split the shared document or the datalet
  • Add notes
  • Import from My Space allows to import a datalet from the private space of user (e.g., My space)
