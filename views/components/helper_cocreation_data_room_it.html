Cocreation Data Room helper
Questa stanza permette al gruppo di utenti di collaborare su di un foglio di lavoro condiviso , per creare un dataset. Il processo è supportato scrivendo note condivise (con un modello che aiuta la comunità a dare tutte le informazioni rilevanti sul processo) e metadati condivisi, che sono pubblicati con il dataset. Inoltre, quando il dataset viene pubblicato per gli altri utenti SPOD, diverse euristiche di convalida vengono utilizzati per mostrare le avvertenze sui campi apparentemente sbagliate. Quando sarà pubblicato, il dataset può essere utilizzato (nel tab “Dataset cocreati” nel primo step della controllet ) da tutti gli utenti SPOD nelle loro discussioni.
1
Features
Caratteristiche della stanza:
  • La scheda Dataset presenta un foglio di calcolo condiviso per inserire i dati, importare ed esportare un documento nel formato CSV.
  • Nella scheda Note si possono inserire note al dataset (ad esempio fonti, bibliografie, etc).
  • Nella scheda Metadata si inseriscono i metadata, importanti per il datasets che saranno pubblicati con il dataset.
  • Una stanza di discussioni e notifiche: Una stanza di discussione è disponibile con notifiche per i messaggi ricevuti e ancora non letti.

2
A quick tour

Una guida veloce per la creazione e pubblicazione di un datatset:

  1. 1. Create a dataset
    1. Creare (o importare) un dataset in collaborazione
    2. Aggiungere note al dataset (ad es. Fonti etc.)
    3. Aggiungere i metadata al dataset (come titolo, descrizione, etichette etc..) per facilitare la ricerca
    4. Il dataset creato può essere visualizzato in un grafico (datalet)
  2. 2.Pubblica il dataset () per tutti gli utenti di SPOD per renderlo disponibile per creare una datalet per discussion nei modi tradizionali ( presente in Novità, nelle stanze di discussione dell’ Agora, etc). Si noti che ogni volta che lo stesso dataset viene pubblicato di nuovo, viene creata una nuova versione.
