Cocreation Data Room helper
This room allows members to collaborate on a shared spreadsheets to create dataset. The process is supported by writing shared notes (with a template that helps the community to write all the relevant information about the process) and shared metadata, that are published with the dataset. Moreover, when published for other SPOD users, several validation euristics are used to show warnings on apparently wrong fields. When published, the dataset can be used (under “Co-created datasets” tab on the first step of the controllet) by all SPOD users in their discussions.
1
Features
This room features include:
  • Dataset tab allows to write in collaboration the dataset, export and import of CSV files.
  • Notes tab allows to write notes on the dataset (like sources etc.).
  • Metadata tab allows to write metadata that are very important for the dataset that will be also published with the dataset.
  • Discussion room and notifications: in discussion room, a new message is notified with an icon with the counter of number of received and unread message.

A quick tour

A quick tour to create and publish a dataset:

  1. 1. Create a dataset
    1. Create (or import) a dataset in collaboration
    2. Add note on the dataset (like sources etc.)
    3. Add metadata to the dataset (like title, description, tags etc.) to facilitate search and discovery
    4. Show the created dataset with a chart (datalet)
